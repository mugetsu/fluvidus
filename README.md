Fluvidus (Alpha)
=========

Fluvidus *(flowing)* a simple fluid carousel implementation.

  - Fluid
  - Fully Customizable
  - Viola!

Version
-
1.0

Installation
--------------
`$('#fluvidus').fluvidus();`

Options
--------------
*Pager*

`$('#fluvidus').fluvidus({
      pager: false // true; pagination style
  });
`
